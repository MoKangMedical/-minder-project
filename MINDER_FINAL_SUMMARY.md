## 📢 营销策略（情感化路线）

### 核心策略
```
不卖功能，卖情感
不讲技术，讲故事
不做广告，做内容
```

### 渠道策略
```
1. 小红书（主战场）
   - 真实用户故事
   - 念想完成打卡
   - 情感共鸣文案
   - 每天1-2条

2. 抖音（视频传播）
   - 15秒情感短片
   - 用户真实案例
   - KOL合作
   - 每周3-5条

3. 朋友圈（社交裂变）
   - 念想卡片分享
   - 邀请好友得会员
   - 口碑传播
```

### 内容示例
```
小红书文案:
"用Minder记录了给妈妈打电话的念想
今天终于打了，妈妈说她等了好久
感谢Minder，让我没有错过 ♥"

抖音脚本:
镜头1: 忙碌的上班族
镜头2: 打开Minder说"记得给妈妈打电话"
镜头3: 晚上收到温柔的提醒
镜头4: 打电话给妈妈，妈妈很开心
文案: "Minder，让我没有错过重要的人"
```
