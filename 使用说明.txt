## 💡 如何转换为Word文档

### 方法1: 使用在线工具（最简单）
```
1. 访问: https://www.aconvert.com/cn/document/md-to-doc/
2. 上传.txt文件（已经是纯文本）
3. 点击"开始转换"
4. 下载Word文档
```

### 方法2: 复制粘贴到Word
```
1. 打开.txt文件
2. 全选（Command + A）
3. 复制（Command + C）
4. 打开Microsoft Word
5. 粘贴（Command + V）
6. 保存为.docx
```

### 方法3: 使用Typora导出
```
1. 用Typora打开.md文件
2. 菜单 → 文件 → 导出 → Word
3. 保存为.docx
```
