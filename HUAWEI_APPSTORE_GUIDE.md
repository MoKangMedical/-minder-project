## 第1步：注册华为开发者账号

### 1.1 访问华为开发者联盟
```
网址: https://developer.huawei.com/consumer/cn/
```

### 1.2 注册流程
```
1. 点击"注册"
2. 选择"个人开发者"或"企业开发者"
   推荐: 个人开发者（免费，审核快）
3. 填写基本信息
   - 姓名
   - 身份证号
   - 手机号
   - 邮箱
4. 实名认证
   - 上传身份证正反面
   - 人脸识别
5. 等待审核（1-2个工作日）
```

### 1.3 费用
```
个人开发者: 免费
企业开发者: ¥99/年
```

## 第3步：上传APK

### 3.1 APK要求

#### 技术要求
```
格式: APK
签名: 必须签名
最小SDK: API 26 (Android 8.0)
目标SDK: API 34 (Android 14)
大小: <100MB
架构: armeabi-v7a, arm64-v8a
```

#### 签名要求
```
必须使用正式签名
不能使用Debug签名
签名证书有效期>25年
```

#### 混淆要求
```
建议开启代码混淆
保护API Key等敏感信息
```

---

### 3.2 上传流程

```
1. 登录华为开发者后台
2. 进入"应用服务" → "AppGallery Connect"
3. 点击"我的应用" → "新建"
4. 填写应用基本信息
5. 上传APK文件
6. 填写版本信息
7. 上传应用图标和截图
8. 填写应用描述
9. 提交审核
```

## 第4步：提交审核

### 4.1 审核前检查清单

```
□ APK已签名
□ 应用图标符合要求
□ 至少3张截图
□ 应用描述完整
□ 隐私政策URL有效
□ 用户协议URL有效
□ 权限说明清晰
□ 版本信息完整
□ 联系方式正确
```

### 4.2 审核标准

#### 功能审核
```
□ 应用可正常安装
□ 核心功能可用
□ 无闪退、卡顿
□ 权限申请合理
□ 无恶意代码
```

#### 内容审核
```
□ 无违法违规内容
□ 无色情暴力内容
□ 无政治敏感内容
□ 描述与功能一致
□ 无虚假宣传
```

#### 隐私审核
```
□ 隐私政策完整
□ 权限说明清晰
□ 数据收集合规
□ 用户可删除数据
```

---

### 4.3 常见审核问题

#### 问题1: 权限过度申请
```
原因: 申请了不必要的权限
解决: 只申请必需的权限，并说明用途
```

#### 问题2: 隐私政策不完整
```
原因: 隐私政策缺少必要内容
解决: 参考华为模板完善隐私政策
```

#### 问题3: 应用描述不符
```
原因: 描述的功能与实际不符
解决: 确保描述准确，不夸大
```

#### 问题4: 图标不清晰
```
原因: 图标分辨率不够或设计不规范
解决: 使用512x512高清图标
```

## 第5步：等待审核

### 5.1 审核时间
```
正常情况: 1-3个工作日
节假日: 可能延长到5-7天
首次提交: 可能需要3-5天
```

### 5.2 审核状态

```
待审核: 已提交，等待审核
审核中: 正在审核
审核通过: 可以上线
审核不通过: 需要修改后重新提交
```

### 5.3 审核不通过怎么办？

```
1. 查看审核意见
2. 根据意见修改
3. 重新提交审核
4. 如有疑问，联系华为客服
```
