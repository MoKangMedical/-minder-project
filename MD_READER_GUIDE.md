# 📱 Markdown阅读软件推荐

## Mac系统推荐软件

### 1. Typora ⭐⭐⭐⭐⭐ (强烈推荐)
**特点**:
- 所见即所得，最美观
- 支持实时预览
- 支持导出PDF、Word
- 界面简洁优雅

**下载**:
- 官网: https://typora.io/
- 价格: ¥89（一次性买断）
- 免费试用: 15天

**安装方法**:
```bash
# 使用Homebrew安装（推荐）
brew install --cask typora
```

---

### 2. MacDown ⭐⭐⭐⭐ (免费推荐)
**特点**:
- 完全免费
- 左右分屏预览
- 支持导出HTML、PDF

**下载**:
- 官网: https://macdown.uranusjr.com/
- 价格: 免费

**安装方法**:
```bash
# 使用Homebrew安装
brew install --cask macdown
```

---

### 3. VS Code ⭐⭐⭐⭐⭐ (万能工具)
**特点**:
- 完全免费
- 功能强大
- 支持Markdown预览
- 可安装插件增强功能

**下载**:
- 官网: https://code.visualstudio.com/
- 价格: 免费

**推荐插件**:
- Markdown All in One
- Markdown Preview Enhanced

---

### 4. 系统自带预览 ⭐⭐⭐ (最简单)
**使用方法**:
1. 右键点击.md文件
2. 选择"打开方式" → "预览"
3. 或直接按空格键快速预览

---

## 🚀 快速安装Typora（推荐）

### 方法1: 使用Homebrew（推荐）
```bash
# 1. 安装Homebrew（如果没有）
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

# 2. 安装Typora
brew install --cask typora
```

### 方法2: 直接下载
```
1. 访问 https://typora.io/
2. 点击 Download
3. 选择 macOS 版本
4. 下载后双击安装
```

---

## 📖 使用建议

### 如果你只是阅读文档
**推荐**: MacDown（免费）或系统自带预览

### 如果你需要编辑和导出
**推荐**: Typora（最美观）或VS Code（免费强大）

### 如果你预算有限
**推荐**: MacDown（免费）或VS Code（免费）

---

## ⚡ 立即安装（最简单方法）

打开终端，复制粘贴以下命令：

```bash
# 安装免费的MacDown
brew install --cask macdown
```

安装完成后，双击任何.md文件即可打开阅读！

---

**推荐**: 先用免费的MacDown试试，如果喜欢再购买Typora
