# UI设计文档
## VoiceReminder AI - 界面设计规范

**版本**: v1.0  
**日期**: 2026-02-08

---

## 1. 设计理念

### 1.1 核心原则
- **极简主义**: 减少视觉干扰，聚焦核心功能
- **一键操作**: 最重要的功能只需一次点击
- **即时反馈**: 每个操作都有清晰的视觉反馈
- **温暖友好**: 使用柔和的配色和流畅的动画

### 1.2 设计目标
- 新用户3秒内理解如何使用
- 核心流程不超过3步
- 支持单手操作
- 适配各种屏幕尺寸

## 6. 动画规范

### 6.1 过渡动画

```
页面切换:
- 时长: 300ms
- 曲线: cubic-bezier(0.4, 0.0, 0.2, 1)
- 效果: 淡入淡出 + 轻微位移

元素出现:
- 时长: 200ms
- 曲线: ease-out
- 效果: 从下往上滑入 + 淡入

元素消失:
- 时长: 150ms
- 曲线: ease-in
- 效果: 淡出
```

### 6.2 交互动画

```
按钮点击:
- 时长: 100ms
- 效果: scale 0.98

卡片点击:
- 时长: 150ms
- 效果: scale 0.98 + 阴影加深

滑动操作:
- 时长: 250ms
- 曲线: ease-out
- 效果: 跟随手指滑动
```

### 6.3 加载动画

```
麦克风波形:
- 3-5条竖线
- 高度随音量变化
- 颜色: #4A90E2
- 刷新率: 60fps

AI思考:
- 3个点依次闪烁
- 间隔: 200ms
- 颜色: #4A90E2
```
