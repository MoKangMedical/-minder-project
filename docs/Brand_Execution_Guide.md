## 📢 营销策略

### 核心策略：情感化营销

#### 1. 小红书（主战场）
```

示例文案:
"用念念记录了给妈妈打电话的念想
今天终于打了，妈妈说她等了好久
感谢念念，让我没有错过 ♥"
```

#### 2. 抖音（视频传播）
```
视频类型:
- 15秒使用演示
- 用户真实故事
- 情感短片

发布频率: 每周3-5条

脚本示例:
镜头1: 忙碌的上班族

镜头3: 晚上收到提醒
镜头4: 打电话给妈妈，妈妈很开心
文案: "念念，让我没有错过重要的人"
```

#### 3. 朋友圈（社交裂变）
```
分享功能:
- 念想卡片（完成后生成）
- 念想统计（月度总结）
- 念想日历（可视化）

裂变机制:
- 分享得3天会员
- 邀请好友得1周会员
- 情侣双人会员优惠
```
