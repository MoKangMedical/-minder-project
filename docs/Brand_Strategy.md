## 💡 品牌命名：念念

### 为什么叫"念念"？

**"念"的三层含义**:
1. **念想** - 心中的想法和愿望
2. **惦念** - 对重要事情的牵挂
3. **念念不忘** - 那些不想错过的时刻

**品牌故事**:
> 每个人心中都有无数个念想：
> - 想给妈妈打个电话
> - 想完成那个一直拖延的项目
> - 想在纪念日给TA一个惊喜
> 
> 但生活太忙，这些念想常常被遗忘。
> 
> 念念，就是你的第二记忆。
> 说出你的念想，AI帮你记住。
> 在合适的时候，轻轻提醒你。
> 
> **让每一个念想，都不被辜负。**

## 🎭 情感化设计细节

### 1. 微交互设计

**心跳动画**:
- 录音时：心形按钮随声音大小跳动
- 等待时：柔和的呼吸效果
- 完成时：心形放大并闪烁

**声音设计**:
- 开始录音：温柔的"叮"声
- 录音中：柔和的环境音
- 完成：温暖的"叮咚"声
- 提醒：柔和的铃声（可自定义）

**触觉反馈**:
- 长按录音：轻微震动
- 完成念想：满足感震动
- 删除念想：警示震动

### 2. 情感化提醒

**不同时段的问候**:
```
早上 (6:00-11:00):  "早安，[用户名] 👋 今天有X个念想等待你"
中午 (11:00-14:00): "午安，[用户名] 🌤️ 记得休息一下"
下午 (14:00-18:00): "下午好，[用户名] ☕ 继续加油"
晚上 (18:00-23:00): "晚上好，[用户名] 🌙 今天完成了X个念想"
深夜 (23:00-6:00):  "夜深了，[用户名] 💤 早点休息"
```

**完成念想的鼓励**:
```
完成1个：  "♥ 又完成了一个念想"
完成5个：  "🎉 今天真棒！完成了5个念想"
完成10个： "🌟 太厉害了！10个念想都完成了"
连续7天：  "🔥 坚持7天了！你真的很棒"
```

**未完成的温柔提醒**:
```
延迟1小时： "💭 [念想]还记得吗？"
延迟1天：   "🌸 [念想]要不要现在做？"
延迟3天：   "💝 [念想]还在等你哦"
```

### 3. 个性化昵称

**AI给用户的称呼**:
```
默认：      [用户名]
亲密模式：  小[姓]、[名]宝
正式模式：  [姓]先生/女士
```
