# 💖 念念Minder - 情感化交互设计手册

## Emotional Interaction Design Manual

**版本**: v1.0  
**创建日期**: 2026-02-08  
**核心理念**: 每一次交互都是一次温暖的陪伴

---

## 🎭 情感化设计原则

### 1. 温暖陪伴，而非冰冷工具
```
❌ 错误: "操作成功"
✅ 正确: "✨ 我会帮你记住这个念想"

❌ 错误: "删除成功"
✅ 正确: "好的，已经帮你忘记了"

❌ 错误: "错误：网络连接失败"
✅ 正确: "网络好像有点问题，稍后再试试吧 🌐"
```

### 2. 鼓励成长，而非简单记录
```
完成1个念想: "开始了！继续加油 ✨"
完成5个念想: "真棒！你越来越棒了 🌟"
完成10个念想: "太厉害了！坚持就是胜利 💪"
完成30个念想: "你已经完成了30个念想！你真的很优秀 ♥"
连续7天: "连续7天都在进步，你真的很自律 🎯"
```

### 3. 理解情绪，而非机械提醒
```
工作日早上: "早安 ☀️ 新的一天，新的开始"
工作日晚上: "辛苦了一天，记得休息 🌙"
周末早上: "周末愉快！今天想做什么呢？"
深夜: "夜深了，早点休息，明天继续加油 💤"
```

### 3. 微交互动画库

#### 点赞动画（完成念想时）
```
效果: 心形从小到大，带旋转
时长: 0.6秒
缓动: spring(stiffness=200, damping=10)

实现:
1. 心形从0.5倍放大到1.2倍
2. 同时旋转15度
3. 回弹到1.0倍
4. 周围出现小星星粒子效果
```

#### 删除动画（删除念想时）
```
效果: 卡片向右滑出，淡出
时长: 0.4秒
缓动: ease-in

实现:
1. 卡片向右平移200dp
2. 同时透明度从1.0到0.0
3. 其他卡片向上平滑填补空位
```

#### 添加动画（新建念想时）
```
效果: 卡片从底部滑入
时长: 0.5秒
缓动: ease-out

实现:
1. 卡片从屏幕底部+100dp位置
2. 滑动到目标位置
3. 同时透明度从0.0到1.0
4. 轻微的弹跳效果
```

```

音调: 低音（800Hz）
时长: 0.3秒
音量: 60%
情感: 确认、安心
```

#### 完成念想
```
音效: 清脆的"叮铃"声
音调: 中高音（1000Hz → 1200Hz）
时长: 0.5秒
音量: 70%
情感: 成就、喜悦
```

#### AI解析成功
```
音效: 柔和的"嗡"声
音调: 渐变（600Hz → 800Hz）
时长: 0.4秒
音量: 50%
情感: 智能、可靠
```

#### 提醒通知
```
音效: 温柔的"铃铛"声
音调: 双音（800Hz + 1000Hz）
时长: 1.0秒
音量: 80%
情感: 温柔、不打扰
```

### 2. 震动反馈（Haptic Feedback）

#### 轻触反馈
```
场景: 点击按钮、切换标签
强度: 轻（Light）
时长: 10ms
模式: 单次
```

#### 成功反馈
```
场景: 完成念想、创建成功
强度: 中（Medium）
时长: 20ms
模式: 双击（间隔50ms）
```

#### 警告反馈
```
场景: 删除操作、错误提示
强度: 重（Heavy）
时长: 30ms
模式: 三连击（间隔30ms）
```

#### 长按反馈
```
场景: 长按录音按钮
强度: 轻（Light）
时长: 15ms
模式: 持续震动（每200ms一次）
```

### 1. 智能问候语

#### 时段问候
```kotlin
fun getGreeting(hour: Int, userName: String): String {
    return when (hour) {
        in 5..8 -> "早安，$userName ☀️\n美好的一天从现在开始"
        in 9..11 -> "上午好，$userName 💼\n今天要完成哪些念想呢？"
        in 12..13 -> "午安，$userName 🍱\n记得好好休息"
        in 14..17 -> "下午好，$userName ☕\n继续加油！"
        in 18..22 -> "晚上好，$userName 🌙\n辛苦了一天"
        in 23..24, in 0..4 -> "夜深了，$userName 💤\n早点休息，明天继续"
        else -> "你好，$userName 👋"
    }
}
```

### 2. 动态鼓励语

#### 完成念想时
```kotlin
fun getCompletionMessage(count: Int, streak: Int): String {
    return when {
        count == 1 -> "开始了！第一个念想完成 ✨"
        count == 5 -> "真棒！已经完成5个念想了 🌟"
        count == 10 -> "太厉害了！10个念想达成 💪"
        count == 30 -> "你已经完成了30个念想！你真的很优秀 ♥"
        count == 50 -> "50个念想！你是念想达人 🏆"
        count == 100 -> "100个念想！你太棒了！🎉"
        count % 10 == 0 -> "又完成了一个念想 ♥ 继续加油"
        streak >= 7 -> "连续${streak}天都在进步，你真的很自律 🎯"
        else -> "又完成了一个念想 ♥ 真棒！"
    }
}
```
