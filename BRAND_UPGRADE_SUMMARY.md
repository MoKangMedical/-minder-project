## 📢 营销策略

### 核心策略：情感化营销

#### 1. 小红书（主战场）
```
内容: 真实用户故事
频率: 每天1-2条
风格: 情感共鸣

示例:
"用念念记录了给妈妈打电话的念想
今天终于打了，妈妈说她等了好久
感谢念念，让我没有错过 ♥"
```

#### 2. 抖音（视频传播）
```
内容: 15秒使用演示 + 情感故事
频率: 每周3-5条
风格: 温暖、感人

脚本:
镜头1: 忙碌的上班族
镜头2: 打开念念说"记得给妈妈打电话"
镜头3: 晚上收到提醒
镜头4: 打电话给妈妈，妈妈很开心
文案: "念念，让我没有错过重要的人"
```

#### 3. 朋友圈（社交裂变）
```
功能: 念想卡片分享
机制: 分享得会员
效果: 口碑传播
```
